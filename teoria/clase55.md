### Ejercicios

**1 -** Crea un script para saber si el usuario es root o no
```bash
#!/bin/bash
# Make sure only root can run our script
if [ "$(id -u)" != "0" ]; then
   echo "This script must be run as root" 1>&2
   exit 1
fi
```
